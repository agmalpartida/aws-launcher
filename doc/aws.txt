# README

AWS 服务的命名规则，

通常如果一个服务的名字是 name
那么这个服务的地址有如下规律
homeURL 应该是 aws.amazon.com/name
consoleURL 应该是 console.aws.amazon.com/name

但是也有例外

homeName 跟 consoleName 不一样,(homeName 比 consoleNmae多一横) 这一类icons命名为xx-xx
比如
homeURL -> aws.amazon.com/app-mesh
consoleURL -> console.aws.amazon.com/appmesh

homeURL -> aws.amazon.com/managed-services
consoleURL -> console.aws.amazon.com/managedservices

StepFunctions
homeURL -> aws.amazon.com/step-functions
consoleURL ->  console.aws.amazon.com/states
